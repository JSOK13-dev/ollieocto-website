# Ollie Ocean Adventure Activity Pack

Printable asset:

- `printable-activity-pack.html`

How to export as PDF:

1. Open `printable-activity-pack.html` in a browser.
2. Choose Print.
3. Select Save as PDF.
4. Use Letter paper, portrait orientation, default margins or no margins.
5. Confirm each activity starts on its own page.

Included pages:

- Ollie coloring page
- Stella coloring page
- Spike coloring page
- Rainbow Bubble coloring page
- Ocean maze
- Spot the difference
- Character matching
- Kindness certificate
